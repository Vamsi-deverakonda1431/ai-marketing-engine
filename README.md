# 🚀 AI Marketing Campaign Analyzer

## 🔍 Overview
An AI-powered web application that analyzes marketing campaigns and suggests optimal tone strategies based on platform and audience.

## ⚡ Features
- Campaign analysis using AI
- Tone vs platform validation
- Smart suggestions
- Clean UI with real-time results

## 🛠 Tech Stack

- Backend: FastAPI
- AI API: OpenRouter
